# 🐂 Horn Research Project

A multi-language demo project inspired by animal horns and wildlife biology.

## Features
- Horn growth simulation (Python)
- Wildlife tracker (JavaScript)
- Web interface (HTML/CSS)
- Simple API server (Node.js)
- Structure engine (C++)
- Animal model (Java)
- Length calculator (Go)
- Diagnostics tool (Rust)
- Configuration files (JSON & YAML)

Exploring the strength and structure of animal horns.
